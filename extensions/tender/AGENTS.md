# Tender

Utility extensions for pi-fleet. Each provides a standalone productivity feature.

## Extensions

| Extension | Role |
|-----------|------|
| `improve-prompt/` | Meta-prompting (Alt+M), reasoning level cycle (Alt+R). Status Bar segment |
| `summarize/` | Auto one-line session summary. Status Bar segment |
| `thinking-timer/` | Elapsed-time display for Thinking blocks (Messages zone) |

## Slash Command Domain Assignment

| Extension | Domain | Rationale |
|-----------|--------|-----------|
| `summarize/` | `summary` | Session summary features |
| `improve-prompt/` | `prompt` | Prompt improvement features |
| `thinking-timer/` | `timer` | Thinking timer display features |
